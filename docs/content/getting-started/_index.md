Get up and running with ddb-lib in minutes. Choose your path based on your project setup.

## Prerequisites

Before you begin, ensure you have:

- Node.js >= 18.0.0
- npm or yarn
- AWS account (for DynamoDB)
- Basic TypeScript knowledge
